# Zenith Food Solutions - Official Website

Premium Catering & Hospitality Services since 2011.

![Zenith Logo](https://www.zenithfoodsolutions.com/images/logo.jpg)

## 🌐 Live Preview
The website is optimized for high-performance static hosting on **Vercel**.

## 🚀 Features
- **Modern Responsive Design**: Optimized for Laptops, Tablets, and Mobile devices.
- **Dark Mode Support**: Seamless theme switching with persistent user preference.
- **Sector-Specific Portfolios**: Detailed sections for Healthcare, Education, and Industrial catering.
- **Lead Management**: Profiles of industry veterans B.R. Ramesh and R. Kumaran.
- **Marquee Clients**: Organized directory of official partner logos with interactive "grayscale-to-color" effects.
- **Premium User Experience**: Smooth scroll animations, card-based testimonials, and specialized horizontal service layouts.

## 🛠️ Technology Stack
- **HTML5**: Semantic structure for better SEO.
- **CSS3**: Vanilla CSS with a customized design system using variables.
- **JavaScript**: Native JS for theme toggling, loaders, and scroll animations.
- **Google Fonts**: "Poppins" for a modern, clean institutional look.

## 📦 Deployment on Vercel
This project is a static web application. To deploy:
1. Push these changes to your GitHub repository.
2. Link the repository to your [Vercel Dashboard](https://vercel.com/new).
3. Vercel will automatically detect the static files and deploy them instantly.

## 📁 Project Structure
- `index.html`: Homepage with wide-hero layout.
- `about.html`: Company legacy and leadership profiles.
- `services.html`: Horizontal service offerings and logistics excellence.
- `clients.html`: Marquee partner directory.
- `testimonial.html`: Partner feedback cardbox grid.
- `css/style.css`: Unified, variable-driven design system.
- `js/main.js`: Dark mode and animation logic.
